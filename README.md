# Suzuki Liana 33920-68J ECU Mapping (реверс-инжиниринг)

Определения карт ЭБУ и файл TunerPro XDF для ЭБУ серии 68J Suzuki Liana,
созданные на основе реверс-инжиниринга штатной прошивки (база: 33920-68JC).

---

## 📌 Поддерживаемые ЭБУ

Проект ориентирован на следующие номера блоков:

* 33920-68JA
* 33920-68JB
* 33920-68JC *(базовая версия)*
* 33920-68JD

> ⚠️ Примечание: Все определения в основном основаны на анализе прошивки 68JC.
> Другие версии предполагаются совместимыми, но требуют проверки.

---

## 🔧 Возможности

* ✅ Определения XDF для TunerPro
* ✅ Топливные карты (основные / обогащение / коррекции)
* ✅ Карты угла опережения зажигания
* ✅ Ограничитель оборотов / ограничитель скорости
* ✅ Коррекции по датчикам (IAT, охлаждающая жидкость и др.)
* ✅ Адресация и структура таблиц

---

## 🧠 Методология

Все данные в этом репозитории получены методом реверс-инжиниринга с использованием следующих инструментов:

* Ghidra
* Ручной анализ дизассемблированного кода
* Сопоставление структур памяти ЭБУ
* Анализ и интерпретация кода с помощью ИИ

Карты и параметры были определены с помощью:

* распознавания паттернов
* определения осей (RPM / нагрузка)
* корреляции с поведением двигателя

1. Получите дамп прошивки вашего ЭБУ
2. Откройте его в TunerPro
3. Загрузите предоставленный XDF-файл
